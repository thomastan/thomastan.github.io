A short, somewhat efficient, primality test in C

Welcome to my first GitHub Page. Onto prime numbers!

By intuition, testing whether a given number is prime or not (primality testing), can be more efficiently done by noting that all prime numbers can be expressed as 6*k ± 1 ≤ sqrt(n) for k = 1, 2, 3, ..., while noting at the same time that the converse of this is not true.

Below is a succinct implementation of mine in C99.

#include <stdbool.h>
#include <math.h>  // for sqrt()

bool isPrime(int number) {
    if (number == 0 || number == 1 || number % 2 == 0 || number % 3 == 0)
        return number == 2 || number == 3;

    int j;
    for (j = 5; j <= sqrt(number); j += 6)
        if (number % j == 0 || number % (j + 2) == 0)
            return false;

    return true;
}
