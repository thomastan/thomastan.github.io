Isolated Python environments %%% DRAFT %%%

A polynomial of degree $n$, $P_n(x) = a_n x^n + a_{n-1} x^{n-1} + a_{n-2} x^{n-2} + \ldots + a_0$ can be represented in C as the vector { a[0], a[1], ..., a[n] }. The vector has size $n+1$.

Such a polynomial can be differentiated with a C function operating on the polynomial’s coëfficient vector (here called coeff):

#include <stdlib.h>

void differentiate(float *coeff, int degree) {
    int i;
    for (i = 0; i <= degree - 1; i++)
        coeff[i] = (i + 1) * coeff[i + 1];

    if (degree >= 1) {
        // The polynomial and its coëfficient vector has a new size
        coeff = (float *) realloc(coeff, degree * sizeof(float));

        degree--;
    }
}
