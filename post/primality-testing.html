A short, somewhat efficient, primality test in C

Welcome to my first Scriptogr.am post. Onto prime numbers!

By intuition, testing whether a given number is prime or not (primality testing), can be more efficiently done by noting that all prime numbers (above 2 and 3) can be expressed as $6k \pm 1 \le \sqrt{n}$ for $k = 1, 2, 3, \ldots$.

Below is my succinct implementation in C99. Despite being three times more efficient than a naïve implementation, simplicity has been preserved.

#include <stdbool.h>  // for true, false
#include <math.h>     // for sqrt()

bool isPrime(int number) {
    if (number == 0 || number == 1 || number % 2 == 0 || number % 3 == 0)
        return number == 2 || number == 3;

    int j;
    for (j = 5; j <= sqrt(number); j += 6)
        if (number % j == 0 || number % (j + 2) == 0)
            return false;

    return true;
}
