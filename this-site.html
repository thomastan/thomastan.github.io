This site

I believe in love at first sight. I fell in love the moment I first saw the Wordpress theme “Kubrick”, the default in Wordpress versions 1.5 through 2.9. I can’t remember on which blog I first saw it on, but I remember the first time I saw the blue bar replaced with a green and the Kubrick theme replaced with a new default in Wordpress v3, that I still wanted to keep Kubrick along with all its blue fidelity.

Transplanting to Scriptogr.am

I’m no longer on Wordpress, so I transplanted the blue bar from Kubrick to one of the default Scriptogr.am themes (Linearian). As long as you’re not viewing this site on a smartphone, you’re seeing my custom theme right now. :-)

The most notable difference should be the change of the accent from a red to match Kubrick’s blue colour (yes, Canadian/Australian 😃).1 Several other improvements have been made, but the end result should be that everything looks “just right”. If you want to know what else has been done to enrich this site, continue reading…

IE6 compatibility

Kubrick was never really IE6 compatible (or was it Wordpress to blame?). With my previous older site, I spent many hours of work to achieve an IE6-compatible correct-looking drop down menu in pure CSS. Good riddance. As Scriptogr.am and the default “Linearian” theme have been written atop HTML5 Boilerplate, things are less of a mess.

Nevertheless, some hacks have been applied while customizing “Linearian” to make this site IE6 backwards compatible. Most notably, the “wrapper” div is now centered.2

I will continue to target CSS3 browsers with this site, but will always test the site in IE6 and hack a fix if anything doesn’t look satisfactory.

Enjoy :-)

Google Code Prettify

I actually prefer SyntaxHighlighter for pretty printing code – its highlighting is more striking. However, Google’s does automatic language detection which is also used by Scriptogr.am to some extent, by default. My prettify stylesheet is a custom one.

MathJax

Another love of mine is LaTeX ^_*.3 I love it in all its glory and can’t live without it, especially the Computer/Latin Modern font. I often find myself struggling to read papers and sites heavily involving mathematics when they aren’t properly typeset using a LaTeX engine. I’ve chose to lead by example and render all math on this site how it should be; beautifully, with MathJax. Here’s some:

By the way, I’m aware my site flies in the face of the typesetting philosophy of using a serif font for the main bodies of text. But we are in the Web 2.0 stage (maybe even later?) of the internet and for me, even in IE6, the result of Kubrick’s sans serif font for main text still looks okay.

Custom fonts

I’ve changed my blog title to be in the “Small Caps” shape of the “Latin Modern” font, did you notice?

Yes, custom fonts are are supported in IE6. Having the custom font appear across different browsers requires having the font in multiple formats: woff (web open font format), eot (embedded open type), ttf (true type), and if you insist, svg (scalable vector graphics).
