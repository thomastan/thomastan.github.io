Basic sorting algorithms

Bubble sort, selection sort and insertion sort all written in C for ascending order1. Kept as minimal as possible so it’s as easy as possible for adaptation to your needs.

Bubble sort

Loop through the list, swapping any adjacent pair of elements if the first is larger. Repeat looping $n - 1$ times (end swapping one place earlier each time).

void bubble(int a[], int n) {
    int i, j, temp;

    for (i = 0; i < n - 1; i++)
        for (j = 0; j < n - 1; j++)
            if (a[j] > a[j + 1])
                temp = a[j], a[j] = a[j + 1], a[j + 1] = temp;
}

Using SSH keys

Using SSH keys

  1. Generate a key pair with ssh-keygen. Place:
    • public key on server: ~/.ssh/authorized_keys.
    • private key on client: ~/.ssh/id_rsa.
  2. Make your private key private (mode 600 = only you the owner can read/write): chmod 600 ~/.ssh/id_rsa.

If you generate a key pair without a passphrase, anybody who captures your private key can access servers with its public key counterpart.
