Engineering

I completed my thesis project with the ECSE department at Monash University1 under the supervision of Dr. Wai Ho Li in the field of computer vision. I accelerated SIFT using OpenMP to thread to multiple cores and used CUDA to unlock the power of the GPU to accelerate SIFT's cousin Affine-SIFT.

View my results below --- and interact! The output is a 3d model obtained from 37 still input images. You need any WebGL capable browser -- that's Chrome, Firefox 4+, Internet Explorer 11 or Safari, no plugin required.

What are you seeing?

Using a feature detector allows "features" of an object to be tracked across several images. Tracking relative distances between each feature and the camera allows the spatial location of each feature to be triangulated and a three-dimensional structure from motion can thus be created.
