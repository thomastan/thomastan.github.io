Math

My honours thesis, "Automated Algorithm Selection for the Traveling Salesman and other $\mathcal{NP}$-hard Problems" involved using evolutionary computing to better analyze and select algorithms in a visual or automated manner.

The approach taken involves using an innovative combination of high-dimensional data analysis, an evolutionary genetic algorithm and a concave approach to polyhedral theory. The result is the visualization of algorithms from which head-to-head algorithm performance is quantified to form an automated solution to the algorithm selection problem.

This page anchors all my shared thesis work, including the published paper and online bite-sized delivery of my thesis as blog posts. For posts I've simply tagged with #math, go here.
